# Skyrim Buildnator

Randomly generate your next Skyrim character build — race, archetype, primary skills, standing stone, combat style, gear, faction, deity, morality, roleplay rules and a challenge modifier.

**Live page:** https://theoaked.github.io/skyrim-buildnator/

## Features

- One-click **Generate Build** rerolls everything.
- 🎲 Reroll any single category.
- 🔒 Lock categories you like — they survive a full reroll.
- **Copy Build** puts a formatted text summary on your clipboard.

## Hosting

Plain static site (HTML/CSS/JS, no build step) served by GitHub Pages from the `main` branch root.
